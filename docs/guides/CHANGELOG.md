### 项目结构

```
Heart-Rythm/
├── backend/            # 后端代码
│   ├── api/           # API层
│   └── data_manager.py # 数据管理模块
├── frontend/           # 前端代码
│   └── templates/      # HTML模板
├── data/               # 数据存储目录
├── docs/               # 文档
├── tests/              # 测试代码
├── main.py             # 主入口文件
└── requirements.txt    # 依赖列表
```

### 如何运行

1. **安装依赖**：
   ```bash
   pip install -r requirements.txt
   ```

2. **启动系统**：
   ```bash
   python main.py
   ```

3. **选择运行选项**：
   - 选项1：启动后端服务（默认端口：8000）
   - 选项2：启动前端应用（默认端口：5000）
   - 选项3：运行示例程序
   - 选项4：运行测试
   - 选项0：退出系统

### 主要API端点

- **情侣管理**：`/couples/`
- **奖励管理**：`/rewards/`
- **积分变动**：`/points/`
- **兑换记录**：`/exchanges/`
- **系统统计**：`/stats/`
- **备份管理**：`/backups/`
