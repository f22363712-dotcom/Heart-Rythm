## 🚀 快速开始（3步）

### 1️⃣ 验证配置
```bash
python scripts/verify_pwa.py
```
**预期**: ✅ 所有检查通过 (6/6)

### 2️⃣ 启动服务
```bash
start_pwa.bat
```
**访问**: http://localhost:5000

### 3️⃣ 安装PWA
- **桌面**: 点击地址栏安装图标 ➕
- **Android**: 点击"立即安装"横幅
- **iOS**: Safari → 分享 → 添加到主屏幕
